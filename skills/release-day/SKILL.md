# Release Day

## When to use

Jeff says something like:
- "I'm dropping a new track on [date]"
- "Stalemate has a new release coming"
- "Just put a song on Spotify, help me announce"
- "Release day for [song title]"

## Process

### Step 1 — Gather release details

In one batched message, ask Jeff for:
1. **Song/album title**
2. **Artist** (Jeff solo / Stalemate / Rad Dad / collaboration)
3. **Release date** — today, future scheduled, already live?
4. **Platforms live on** — Apple Music, Spotify, YouTube Music, SoundCloud, Bandcamp, etc.
5. **One-line "what it's about"** — helps write the copy
6. **Any specific hook** — first release in X years, recorded at home, features Y, etc.
7. **Who's in it** — band members, collaborators, engineer credits

If the release is tied to a specific song in `memory/music/song-catalog.md` or `stalemate-songlist.md`, pull the existing context — Jeff won't need to re-explain.

### Step 2 — Produce the bundle

Generate each of these as draft blocks Jeff can copy-paste (label each clearly):

**A. Instagram post caption** (< 220 characters for main feed; include hashtags like #newmusic #indie #{genre})
**B. Instagram story text overlay** (short hype line)
**C. Facebook post** (longer, more context, friends-and-family tone)
**D. X / Twitter post** (< 280 characters)
**E. Newsletter blurb** (if Jeff has an audience list — 150–200 words)
**F. raddadband.com news entry** (if relevant — markdown formatted)
**G. Bandmates group-text** (via `/band-comms` style — celebratory, asks them to share)
**H. Streaming links checklist** (Apple Music / Spotify / YouTube Music / SoundCloud — Jeff pastes URLs for each)

### Step 3 — Propose a 72-hour post schedule

Offer a timed posting schedule Jeff can follow:

- **T-0 (release day morning):** IG main feed + story + X + Facebook
- **T+1 day:** Behind-the-scenes video or photo
- **T+2 days:** "If you missed it…" follow-up with review quote / listener reaction
- **T+3 days:** Playlist add ask / save-for-later reminder
- **T+1 week:** Lyric breakdown or acoustic cover post

### Step 4 — Update memory

Once Jeff confirms the release is live:
- Update `memory/music/discography.md` with the new release under the right artist section
- Mark the corresponding song in `memory/music/song-catalog.md` as `released`

### Step 5 — Schedule follow-ups

Offer: "Want me to schedule a T+7 'how's it performing' check-in so we can plan the next push?"

## Platform copy styles (for voice consistency)

- **Instagram:** Lower case, short sentences, one emoji max, hashtag cluster at end
- **Facebook:** Warmer, longer, tags bandmates, a "you know me" voice
- **X:** Punchy, self-aware, often a joke
- **Newsletter:** Full thoughts, like a letter — "here's the story of this song"
- **Bandmates text:** "Yo — dropped. Share it if you feel it. Thanks for being in this with me."

## Never do

- Never post to any platform directly.
- Never invent quotes, reviews, or listener counts.
- Never draft as Rad Dad announcing a streaming release unless Rad Dad actually has streaming releases — as of Apr 2026, Rad Dad has none (only Stalemate and Jeff Story solo are on streaming).
