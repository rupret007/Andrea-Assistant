# Market Scan — Music

Like `/intel-brief` but focused on music industry signal that affects Jeff's artist projects.

## Standard scan areas

1. **Indie pop-punk / punk scene** — new releases, playlist adds, algorithm shifts, emerging acts (for Rad Dad reference)
2. **Indie alt / melodic rock** — for Stalemate + solo positioning
3. **Apple Music / Spotify / YouTube Music indie artist tools** — what's newly available, pricing changes, editorial opportunities
4. **Texas live-music scene** — festivals, venues, booking trends (if Jeff is in Texas — verify in memory)
5. **Home-recording / production tools** — plugins, free sample packs, new DAWs Jeff could use
6. **Sync licensing opportunities** — companies looking for indie songs

## When to use

- "Music market scan"
- "Any music news"
- "What's happening in pop-punk"
- Before a release-day (complements `/release-day`)
- Before booking a tour / show run
- Monthly music-industry ritual

## Process

### Step 1 — Scope

Default = all 6 areas. Or Jeff can narrow ("just playlist trends" / "just sync licensing").

### Step 2 — Run Exa searches

Examples:
- `"indie pop-punk new releases playlist trends April 2026"`
- `"Spotify editorial playlist pitching independent artist best practices 2026"`
- `"Texas music festival lineups announced for summer 2026"`
- `"sync licensing opportunities indie musicians 2026"`
- `"home recording tools new plugin releases 2026"`

### Step 3 — Cross-reference with Jeff's catalog

Pull `memory/music/discography.md`, `song-catalog.md`, `stalemate-songlist.md`, `rad-dad-setlist.md` — mention when a trend, opportunity, or reference artist overlaps with a specific Jeff song or band.

### Step 4 — Format the scan

```
🎵 **Music Market Scan — {month}**

## Trends that matter for you
- **{insight}** — {why it matters for your work}
  [source]({URL})

## Playlist / editorial opportunities
- {specific, concrete}

## Similar-artist activity (cues for Rad Dad / Stalemate)
- {what {referenced band} is doing that you could learn from}

## Sync / licensing opportunities
- {concrete opportunities you could pitch}

## Tech / production tools worth a look
- {specific tools that fit your stack}

## Action candidates
- {2–3 concrete moves you could make this week}
```

### Step 5 — Offer next steps

- "Want me to /mission a specific play (e.g., 'pitch 3 sync libraries this month')?"
- "Want me to add any of these to your watchlist?"
- "Want me to draft a submission for {opportunity}?"

## Never do

- Never impersonate a label / manager / PR person in outreach drafts.
- Never claim a track is available for sync if Jeff hasn't confirmed rights.
- Never invent playlist curator contacts — only surface verifiable public info.
