# Rehearsal Proposal

## When to use

Jeff says something like:
- "Propose Rad Dad practice times for next week"
- "Find us a rehearsal slot"
- "When should we practice before the show"
- "Set up a Stalemate rehearsal" (remember — Stalemate is paused while Sean is on paternity leave; ask before proposing)

## Process

### Step 1 — Confirm band + goals

Ask in one message:
1. **Which band?** Rad Dad / Stalemate
2. **What's the goal of this rehearsal?** (general upkeep, specific show prep, working new song, full run-through)
3. **Rough window?** (next week, this weekend, next Tuesday-Thursday)
4. **Duration target?** (90 min / 2 hours / 3 hours)
5. **Anything off-limits?** (dates band members can't do that Jeff already knows)

If the band is Stalemate and Sean is still on paternity leave, ask Jeff whether Sean's available before going further.

### Step 2 — Pull Jeff's calendar

Use Google Calendar `list_events` for the proposed window. Identify blocks that are:
- Free for 2+ hours
- Evenings (weekdays 6pm+) or weekend afternoons work best for typical rehearsal culture
- Not colliding with family commitments or known Cisco meetings

### Step 3 — Propose 2–3 slot options

Produce slot candidates scored by:
- Jeff's calendar openness
- Typical band availability (weekday evening, weekend afternoon)
- Proximity to any looming gig (closer to show = more leverage)

Format:

```
🎸 **Rehearsal options for {band} — {window}**

**Option 1: {day, date, start–end time}**
- {Why this works — e.g., "Jeff clear, 4 days before Wildflower"}
- Focus: {specific setlist work — e.g., "lock down Breed, Linoleum, Drinking Song"}

**Option 2: {same}**
**Option 3: {same}**
```

### Step 4 — Propose a focus

For each option, suggest what to rehearse based on:
- Upcoming shows from `memory/projects/rad-dad.md`
- Setlist from `memory/music/rad-dad-setlist.md` or `stalemate-songlist.md`
- Any songs noted as "needs work" or "new"

Prioritize: songs that haven't been played in a few weeks, songs with known rough spots, new additions.

### Step 5 — Draft the group-text

Once Jeff picks a direction, offer to draft the group-text via the style from `/band-comms`:

```
Yo — practice {day} {time} at {usual spot}. Focus: {setlist items}. Who's in?
```

### Step 6 — Hand off

Return the draft for Jeff to copy-paste into his group-text app. Do NOT send anything directly.

Offer: "Want me to add a tentative event to your Google Calendar for the preferred option?" (with approval).

## Never do

- Never add events to calendar without explicit approval.
- Never send the group-text — draft only.
- Never assume availability for Travis, Lucky, Che, or Sean — their responses go through Jeff.
- Never propose a rehearsal for Stalemate while Sean is on paternity leave unless Jeff explicitly says he's available.
