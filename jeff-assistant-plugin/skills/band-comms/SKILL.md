# Band Comms

## When to use
Jeff asks for help writing a message to his bandmates — confirming practice, sharing a setlist draft, announcing a show, coordinating logistics, or nudging the band on a decision.

## Core constraint
**Draft-only. Never send.** Output the message, let Jeff copy/paste it into his messaging app. Do not attempt to send via any connector on Jeff's behalf.

## Process

1. **Which band?** If Jeff didn't say, ask: "Rad Dad or Stalemate?"
2. **What's the message?** Identify the purpose — practice, show, setlist, gear, scheduling, social.
3. **Who's the audience?**
   - Default = all current active members.
   - If the message is Stalemate-related, confirm with Jeff whether to include Sean given his paternity leave.
4. **Gather any specifics Jeff has already given you.** If not enough to draft, ask one focused follow-up — no more than one question at a time.
5. **Draft the message** in the style of a group text between guys who've been friends since 1997. Casual, brief, direct.
6. **Return the draft** in a quoted block so Jeff can copy-paste cleanly.
7. **Offer variations** if useful (e.g., "want a friendlier version or a more urgent one?").

## Style guide

- **Tone:** Casual, band-brothers. These guys have been playing together for almost 30 years. Never formal.
- **Length:** 1–4 sentences unless it's a full show-day brief.
- **Structure:** Lead with the point, end with a clear ask or yes/no question.
- **No corporate-speak.** No "touching base" or "circling back."
- **Use names, not roles.** "Hey Trav" or "Yo Che" beats "Hey lead guitarist."
- **Emojis OK in moderation** (🎸 🥁 🎤) if it's celebratory or show-related.

## Examples

### Practice confirmation
> Yo — practice this week. Thinking Thursday, 7pm at our usual. Saturday if Thursday doesn't work. Who's in for which?

### Setlist proposal (Wildflower show)
> Wildflower setlist draft for the 16th:
> 1. [Opener]
> 2. [Song 2]
> ...
> Sound good? Want to swap anything?

### Gig announcement (to socials via Jeff)
> 🎸 RAD DAD — May 16 at the Richardson Wildflower Festival. Come hang. Details at raddadband.com.

### Nudge on a slow response
> Hey — need a yes/no on [thing] by [day] so I can [reason]. What's the word?

## Never do

- Never send anything. Draft-only.
- Never draft practice messages to Sean while he's on paternity leave (unless Jeff says he's back).
- Never invent commitments from other members ("Travis said yes") without Jeff confirming.
- Never copy private band context into any file or connector without asking.
