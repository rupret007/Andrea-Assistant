# Episodes — what happened, logged by month

Time-stamped log of what Andrea observed or did, month by month. Feeds into memory consolidation — repeated episode themes promote to semantic facts (in `memory/people/`, `memory/projects/`) or procedural playbooks.

## Structure

```
episodes/
├── 2026-04/
│   ├── 2026-04-17-andrea-v1-v3-shipped.md
│   └── 2026-04-18-conductor-pattern-added.md
├── 2026-05/
│   └── ...
└── README.md
```

## What counts as an episode

- A significant interaction Andrea had with Jeff
- Output of a scheduled ritual
- An Action Bundle with non-trivial outcome
- A meeting that happened
- A creative session (lyric capture, coding block, band practice)

## What does NOT belong here

- Routine briefings Jeff scanned and discarded
- Tool-call minutiae
- Anything below the "worth remembering" threshold

## Cadence

- Scheduled tasks auto-append to current month when they complete
- Nightly consolidation reviews last 7 days' episodes for patterns

## Template

```markdown
# {YYYY-MM-DD} — {episode title}

**Participants:** {Jeff + anyone else involved}
**Source:** {which ritual / skill / manual conversation}

## What happened
{2-5 sentences, honest, first-person from Andrea where natural}

## What changed
{memory updates, decisions made, commitments taken}

## Open threads
{anything ongoing this episode touched}
```
